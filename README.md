# PortfolioWeb

PortfolioWeb is a simple Express.js-based portfolio site.

## Setup

1. Copy `.env.example` to `.env`.
2. Fill out your credentials.
3. Use `npm start` for local development, or see `DEPLOY.md` for deployment.

## Features

- Express.js
- EJS templating
- Email via SMTP
